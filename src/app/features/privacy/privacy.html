Política de privacidad

Call Metrics Logger

Última actualización: 11 de julio de 2026

Call Metrics Logger es una extensión de navegador diseñada para ayudar a equipos de soporte y call center a registrar resultados de llamadas desde un panel lateral del navegador y sincronizar esos registros con el tablero conectado de Call Metrics.

Datos recopilados

La extensión puede recopilar los siguientes datos:

  • Dirección de email, usada únicamente para autenticación mediante Supabase.
  • Datos de sesión de autenticación, usados para mantener la sesión iniciada.
  • Registros de resultados de llamadas creados por el usuario, por ejemplo si una llamada fue normal, derivó en una visita técnica, una reagenda o una instalación.
  • Notas opcionales ingresadas por el usuario.

Datos no recopilados

La extensión no recopila nombres de clientes, números de teléfono, direcciones, números de cuenta, historial de navegación, contenido de sitios web, datos de ubicación, información financiera, información de salud ni comunicaciones personales.

Cómo se usan los datos

Los datos recopilados se usan únicamente para autenticar al usuario y sincronizar registros de llamadas con el tablero conectado de Call Metrics. Esto permite revisar totales diarios, tasas de conversión mensuales y tendencias de rendimiento.

Compartición de datos

Los datos del usuario no se venden, alquilan, transfieren ni comparten con terceros para publicidad, evaluación crediticia ni fines no relacionados con la herramienta.

Almacenamiento y acceso

Los datos de autenticación y los registros de llamadas se almacenan usando Supabase. El acceso a los registros está restringido a usuarios autenticados según las reglas de acceso configuradas en la base de datos.

Control del usuario

Los usuarios pueden crear, editar y eliminar sus propios registros de llamadas. También pueden cerrar sesión en la extensión en cualquier momento.

Contacto

Para preguntas sobre esta extensión o esta política de privacidad, escribí a gabi.antico@gmail.com.

Privacy Policy

Call Metrics Logger

Last updated: July 11, 2026

Call Metrics Logger is a browser extension designed to help support and call-center workers record call outcomes from a browser side panel and sync those records with the connected Call Metrics dashboard.

Data Collected

The extension may collect the following data:

  • Email address, used only for user authentication through Supabase.
  • Authentication session data, used to keep the user signed in.
  • Call outcome records created by the user, such as whether a call was a normal call, a technical visit, a reschedule, or an installation.
  • Optional notes entered by the user.

Data Not Collected

The extension does not collect customer names, phone numbers, addresses, account numbers, browsing history, website content, location data, financial information, health information, or personal communications.

How Data Is Used

The collected data is used only to authenticate the user and sync call records with the connected Call Metrics dashboard. This allows the user to review daily totals, monthly conversion rates, and performance trends.

Data Sharing

User data is not sold, rented, transferred, or shared with third parties for advertising, creditworthiness, or unrelated purposes.

Data Storage And Access

Authentication data and call records are stored using Supabase. Access to records is restricted to authenticated users according to the configured database access rules.

User Control

Users can create, edit, and delete their own call records. Users can also sign out of the extension at any time.

Contact

For questions about this extension or this privacy policy, contact gabi.antico@gmail.com.
